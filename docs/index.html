The agent that did the research

A CrewAI flow, one crew per app, resumable per-app caching. Deterministic facts are never asked of an LLM; disputed fields are settled by a stronger model; verification runs on a different model family so its errors don't correlate with the researcher's.

S0 · DETERMINISTIC
Registry checks

Composio toolkit registry match + official MCP registry lookup + homepage liveness. Pure Python.

Scout → Analyst

S2 · VERIFY ×100
Skeptic Verifier

Every cited evidence page is pre-fetched in Python and injected; a different model lineage grades each field tool-free: confirmed / contradicted / unsupported / dead.

gpt-4o-mini
S3 · FIX
Targeted re-research

Only flagged fields are re-researched with tools; fixed records are verified again. pass-1 vs final kept per field.

S4 · EVAL
Eval layer

Deterministic checks (schema, URL liveness) + a third model lineage judging field↔evidence faithfulness on a stratified sample + human spot-check CSV.

gpt-4.1-mini human
S5–S6 · REPORT
Analysis → this page

All aggregation in Python; this page is rendered from patterns.json / accuracy.json / results.json. No hand-written numbers.

no LLM
Where humans were needed: seeding the 100-app list, API keys, three pipeline problems the agents couldn't see for themselves (gpt-5 models rejecting CrewAI's stop param; a too-loose MCP-registry name matcher that called "storefront" a match for "Front"; a first verification design that let the verifier skip fetching — it burned the OpenRouter budget flagging 70% of correct fields as "unsupported", and was redesigned to pre-fetch evidence in Python and grade tool-free on OpenAI), the human spot-check, and final review of this page's copy. Everything else — research, verification, fixing, scoring — ran unattended.

How we know it's right

500 field-level claims were independently graded; 166 were flagged, and 74 apps had fields re-researched and fixed. The judge (third model family) rated 83% of 100 sampled fields faithful to their cited evidence; a human agreed with —% of 0 sampled claims.
