Where humans were needed: seeding the 100-app list, API keys, three pipeline problems the agents couldn't see for themselves (gpt-5 models rejecting CrewAI's stop param; a too-loose MCP-registry name matcher that called "storefront" a match for "Front"; a first verification design that let the verifier skip fetching — it burned the OpenRouter budget flagging 70% of correct fields as "unsupported", and was redesigned to pre-fetch evidence in Python and grade tool-free on OpenAI), the human spot-check, and final review of this page's copy. Everything else — research, verification, fixing, scoring — ran unattended.
